Anleitung Nutzung GitHub

Example-Repository: https://github.com/informatixwifi/wifisoftwareengineering

Start: Wir bewegen uns über die Kommandozeile mit "cd c:\..." in unser lokales Repository-Verzeichnis.

Nachfolgend die wichtigsten Befehle in der verwendeten Reihenfolge:

# Befehl 1: git clone https://...link_von_repository

_ _ _kopiert das Serververzeichnis in das angegebene lokale Verzeichnis (einmalig erforderlich). Danach sollte man mit "cd" weiter in das Repository-Hauptverzeichnis gehen.

# Befehl 2: git pull

_ _ _holt die aktuelle Version des Server-Verzeichnisses.

# Befehl 3: git add .

_ _ _dem Index hinzufügen.

# Befehl 4: git commit -m "hier Kommentar einfügen"

_ _ _gibt die Dateien im Verzeichnis "frei" bzw. aktualisiert sie lokal. Mit [-m "..."] kann man einen Kommentar hinzufügen.

# Befehl 5: git push origin master

_ _ _lädt die geänderten lokalen Dateien wieder auf den Server.
